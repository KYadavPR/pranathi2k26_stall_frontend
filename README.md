# 🔱 Soda Shastra Frontend - Kiosk & POS

The premium, interactive interface for the **Soda Shastra POS**, designed for a seamless customer kiosk and administrative management experience during **PRANATHI 2K26**.

## ✨ Features
- **Tap-to-Order Kiosk**: A vibrant, mobile-friendly interface ("Divine Tickets") for customers to browse and order.
- **Admin Dashboard**: Real-time sales analytics, profit tracking, and inventory management with glassmorphism aesthetics.
- **Smart Bundle Builder**: Advanced UI for creating product combos with automated cost calculation.
- **Thermal Receipt System**: Standardized billing layout optimized for 3-inch (80mm) thermal printers.
- **Laptop Optimized**: Desktop-first layout enhancements for professional stall management.

## 🎨 Design System
- **Theme**: Ancient Taste, Modern Twist.
- **Palette**: Deep Maroon (#1e0707), Divine Gold (#d4af37), and Vibrant Crimson.
- **Typography**: Cinematic serif headers paired with clean, readable body fonts.

## 🛠️ Tech Stack
- **React + Vite**: High-performance frontend framework.
- **Axios**: Promised-based HTTP client for API communication.
- **Vanilla CSS**: Custom-crafted design tokens and animations.
- **React Router**: Seamless navigation between Kiosk and Admin views.

## 📦 Setup & Deployment

1. **Clone the repository**:
   ```bash
   git clone <your-repo-url>
   cd frontend
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure API Endpoint**:
   Update `src/api.js` to point to your live backend:
   ```javascript
   const API = "https://your-backend-on-render.com/api";
   ```

4. **Launch Development Server**:
   ```bash
   npm run dev
   ```

5. **Build for Production**:
   ```bash
   npm run build
   ```
